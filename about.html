About Me

I'm a Web Developer

Saya adalah seorang mahasiswa yang mengambil jurusan Teknik Informatika di Universitas Bengkulu. Dalam perjalanan akademik ini, saya merasa terus-menerus terpesona oleh dunia teknologi yang dinamis dan potensinya yang tidak terbatas dalam membentuk masa depan. Ketertarikan saya pada pemrograman semakin berkembang seiring berjalannya waktu, dan saya tertarik pada aspek pemecahan masalah kreatif dalam pengembangan perangkat lunak.
Selama studi, saya berkomitmen untuk menguasai berbagai bahasa pemrograman, mulai dari bahasa dasar seperti C++ dan Python hingga teknologi pengembangan web modern seperti HTML, CSS, JavaScript, dan framework seperti React. Saya selalu berupaya memperluas pengetahuan saya tentang teknologi mutakhir, termasuk pembelajaran mesin, kecerdasan buatan, dan analisis data. Dengan terus mengikuti tren terbaru, saya berharap dapat mengembangkan keterampilan yang fleksibel sehingga saya dapat menghadapi berbagai tantangan di bidang teknologi informasi.
Saat saya mendalami area ini, fokus saya adalah mengasah kemampuan, tidak hanya untuk kesuksesan akademis, tetapi juga untuk memberikan dampak nyata di dunia. Semangat belajar dan dorongan untuk berinovasi menginspirasi saya untuk berkontribusi secara bermakna di industri teknologi. Baik melalui pengembangan perangkat lunak, analisis data, atau solusi digital, saya bertekad untuk mencapai kesuksesan di bidang yang menarik dan terus berkembang ini.
